RepSaga — Material + Saga (Three Clean Directions)

Zero pixel art. Three takes on “Material Design with RPG soul”. All three share the same gamification features (LVL badge, XP bar, rank, streak) and the same components — only font, palette, icon style, and shape language differ. Pick one and we’ll build it.

How to choose

A — Midnight Arcana · Highest probability of feeling “premium & mystical” without novelty fatigue. Broad appeal, works for 25–45 demo, aligns with the `arcane` purple-gold palette already shipped. Recommended if you want something elegant that ages well.

B — Parchment Saga · Most distinctive in the market — no other fitness app is light-warm. Strongest “RepSaga is a tabletop RPG” differentiator. Risk: cream-beige reads “vintage” to some users; less popular for gym use (glare in bright environments). Recommended if you want to stand out most.

C — Tactical Forge · Closest to what modern esports athletes expect. Highest information density per screen. Risk: orange-copper primary shifts the brand away from purple-first; feels more “gaming” than “fitness.” Recommended if your core user thinks of the gym as training for combat.

All three use the same gamification primitives, the same component set, and Flutter’s built-in icon system where possible. Reply with A, B, or C and I’ll draft the full spec + theme rewrite plan.
