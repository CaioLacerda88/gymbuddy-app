17.0 scope & risk 3–4× current — new pixel font, asset pipeline, icon replacement sweep, pixel-grid layout discipline. +30%. New pixel-asset pipeline + pixel-rendering widget; modern chrome stays intact. ±0 — 17.0 as spec'd plus one pixel hero art asset.
Accessibility (large text, i18n) Fragile — bitmap fonts don't scale; pt-BR accented chars need a custom glyph set. Safe — Inter handles everything; pixel text is confined to short labels like "LEVEL UP". Safe.
Paywall pitch (Phase 16b) Strongest — "this is clearly not Hevy" in a single screenshot. Strong — character avatar + celebration overlay carry the pitch. Weak — nothing to show that Hevy doesn't.
Re-work of merged PLAN.md Heavy — rewrite 17.0, Milestone Signal section, most Anti-Patterns (badges/grids allowed again). Targeted — rewrite 17.0 to introduce pixel-asset pipeline; amend Milestone Signal to allow curated pixel artwork inside the vertical cap. Minimal — one line change to 17.0.
Brand / app-icon pick Option 1 (sword) or 8 (knight lifting). Option 2 (dumbbell banner) or 9 (R + laurel). Option 5 (clean R + dumbbells) — closest to current chrome.

Notes: pixel sprites are hand-drawn SVGs rendered crisp via shape-rendering=crispEdges — they approximate the reference but are not final artwork. A real implementation would use PNG sprites at 1× and 2× with FilterQuality.none in Flutter, or an atlas loaded via flutter_svg. Fonts used: Press Start 2P + VT323 for pixel text, Inter for modern text.
