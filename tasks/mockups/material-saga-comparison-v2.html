Shared Rules — Apply to whichever direction wins

  • Reward-color scarcity: the bright accent (copper / gold / amber) is PR-and-level-up ONLY. If it's decorating weekday UI, the system is broken.
  • No Cinzel / Cormorant: PO confirmed they fail fatigue-state reading at 15px-and-below. Always Barlow Condensed, Rajdhani, or Space Grotesk for display.
  • Body type is non-negotiable: Inter or IBM Plex. Never use the display font for paragraph copy.
  • Lift icon rule: Side-view barbell with asymmetric rectangle plates. No circles-on-stick (that's a dumbbell). No generic gym emoji.

PO Signal (retention psychology)

  • Identity > stimulant — daily motivation color must feel "in progress," not "hyped." Muted bronze / deep violet / slate cobalt all work; saturated red/orange across-the-board does not.
  • Variable-ratio reward — the gold/copper/amber flash must feel rare. If users see it every workout, dopamine adapts and the color goes dead.
  • First-week warmth — onboarding and first-completion screens can break the scarcity rule briefly to build attachment.
  • Dark UI default — gym lighting varies wildly; dark surface is lower cognitive load in both bright and dim environments.

UI/UX Signal (anti-generic-AI)

  • Kills: teal-on-navy, 3D dumbbell emoji, gradient buttons with drop shadow, 3-identical stat tiles, Inter-everywhere-at-8dp.
  • Monoline vs two-tone vs solid — pick one icon system per direction and enforce it. Mixing is what reads as "generic fitness app."
  • Typographic rhythm — exactly one display font, one body font, one mono/numeric font. Three families max.
  • Real composition > mood board — if the mockup can't reflect actual Flutter widget tree (LVL → status → hero → buckets → routines), it's lying.

How to pick

Pick A (Forged Path) if RepSaga's brand voice is "you're building something over years." Safest retention play. Hardest to make feel generic — but also the easiest to make feel muted. Needs disciplined copper use.

Pick B (Arcane Ascent) if you want to keep the purple brand equity and pixel DNA. Highest differentiation in the App Store (no one else uses deep violet). Risk: "fantasy RPG vibe" was exactly the signal some users found off-putting in pixel v1.

Pick C (Iron Codex) only if you want a hard reset away from the saga/fantasy framing entirely. Most cross-demographic appeal. Weakest story attachment — the hardest one to build a narrative-driven feature set on top of.

Unanimous recommendation from PO + UI/UX: pick A or B, not C. A is the retention-safe choice; B is the brand-continuity choice. The lift icon in all three uses the side-view-barbell pattern that replaces the badly-drawn v1 icon.
